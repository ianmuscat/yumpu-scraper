# Yumpu scraper

Download raw JPEG files from documents on https://www.yumpu.com

## Build

To build this project you will need Go (https://golang.org/doc/install).

```
go build
```

## Usage

```
Usage of ./yumpu-scraper:
  -id string
        Yumpu document ID
  -out string
        directory where to download jpeg files (default ".")
```
